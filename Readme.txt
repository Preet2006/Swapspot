# Swaspot

**Swaspot** is a web platform that allows users to easily swap items within their local community.

## Features
- List, search, and swap items with others.
- Secure user authentication for safe transactions.
- Real-time notifications for updates on swap requests and item status.
- Simple, user-friendly design for easy navigation.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js
- **Database**: MongoDB

## Installation
1. Clone the repository:
   
   git clone https://github.com/Preet2006/Swapspot.git
2.Navigate to the project folder:
    cd swapot
3.install the necessary dependencies:
   npm install
4.Start the application:
   npm start
5.Open your browser and go to http://localhost:3000.
